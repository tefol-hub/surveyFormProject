# Student Survey Form – Responsive Web Form

This project is a **responsive survey form** built as part of the
[freeCodeCamp Responsive Web Design Certification](https://www.freecodecamp.org/).

It demonstrates the ability to design and implement structured, user-friendly forms using **HTML and CSS**.

---

## 🚀 Live Demo

👉 https://tefol-hub.github.io/student-survey-form/

---

## 💼 Project Highlights

* Built a **fully functional HTML form** with validation
* Implemented multiple input types:

  * Text, email, and number fields
  * Dropdown selections
  * Radio buttons and checkboxes
  * Textarea input
* Applied **form validation constraints** (required fields, min/max values)
* Designed a **clean and responsive UI**
* Structured inputs for **usability and accessibility**

---

## 🛠️ Tech Stack

* HTML5 (forms, semantic structure, validation)
* CSS3 (layout, styling, responsiveness)
